## 🚀 Instalación y Uso

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No se requieren dependencias especiales

### Pasos para Usar

```bash
# 1. Clonar el repositorio
git clone https://github.com/Jhersonva/PORTFOLIO-JHERSON-VILLA.git

# 2. Navegar al directorio
cd PORTFOLIO-JHERSON-VILLA

# 3. Abrir en el navegador
# Opción A: Abrir index.html directamente
# Opción B: Usar un servidor local
python -m http.server 8000
# Luego ir a: http://localhost:8000
```
